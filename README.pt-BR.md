# Image Gallery

Uma galeria de imagens desenvolvida com Bootstrap, jQuery e Webpack para praticar conceitos de desenvolvimento front-end moderno.

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- jQuery
- Webpack

## Funcionalidades

- Layout responsivo
- Grid de imagens
- Modal para visualização
- Filtro de categorias
- Organização modular com Webpack

## Estrutura

- Modularização do JavaScript
- Bundling com Webpack
- Estilização com Bootstrap e CSS

## Objetivo

Praticar a integração entre Bootstrap, jQuery e Webpack, aplicando conceitos de organização de projetos, manipulação do DOM e responsividade.
