# Image Gallery

- 🇧🇷 Portuguese version: [README.pt-BR.md](README.pt-BR.md)

A responsive image gallery built with Bootstrap, jQuery, and Webpack.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- jQuery
- Webpack

## Features

- Responsive layout
- Image grid
- Category filters
- Image modal
- Modular architecture with Webpack

## Getting Started

```bash
npm install
npm run dev
